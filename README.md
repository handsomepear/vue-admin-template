# vue-admin-template

积累一套常用的实战技巧 例如：全局组件自动注册 权限控制 路由分区

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
